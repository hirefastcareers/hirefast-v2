Design principles crib (for Cursor)
SURFACES: bg #090d16, surface #0f1522, surface-2 #141d2e, border #1f2d47
BRAND: blue #3b6ef5 (primary), hover #4d7ef6
TRUTH ENGINE (semantic only): emerald safe/match, amber warning, rose risk/reject
TYPOGRAPHY: DM Sans (UI), DM Mono (data, £, match %, postcodes)
RADII: card 14px, input/button 10px, badge 6px, pill 100px
RULES: Max 3 candidate fields, 5 recruiter. Skeleton on all async. Postcode outward only. RTW "(Self-declared)". Apply = Vaul bottom sheet. tabular-nums on all numeric. No dot grids, no Inter, no full postcode, no centred apply modal.
